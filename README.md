# System Estoque Python 
Sistema simples de controle de estoque desenvolvido em Python.

## Funcionalidades 
- Adicionar itens
- Listar itens
- Calcular valor total do estoque

## Conceitos Aplicados 
- Programação Orientada a Objetos(POO): Uso de classe 'ItemEstoque' para representar os itens.
- Listas: Armazenamento de objetos em 'lista_itens'.
- Estruturas de Repetição (for): Para percorrer e exibir itens do estoque.
- Estruturas Condicinais (if/else): Controle de fluxo do menu e validações.
- Funções: Organização de código em 'adicionar_itens' e 'mostrar_itens'.
- Entrada de Dados: Uso de 'input()' intereção com usuário.
